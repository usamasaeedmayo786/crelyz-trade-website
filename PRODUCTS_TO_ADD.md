## Instructions for Adding Products

1. Visit each Amazon URL
2. Extract:
   - Full product name
   - Price (CAD)
   - Product images (main image URL)
   - Description
   - SKU (create one if not available)
3. Use the admin panel at `/admin/products/new` to add products
4. Or use the SQL migration file (see `004_add_amazon_products.sql`)
